# 📚 Best-Selling Books Data Cleaning Project

## 🔍 Project Overview
This project focuses on cleaning and preparing a best-selling books dataset for analysis and modeling.

The raw dataset contained multiple structural and formatting issues that prevented accurate analysis.

---

## 🧹 Key Data Issues Identified

- Price stored as text with currency symbols
- Rating stored as descriptive strings
- Mixed publishing date formats
- Reading age stored as text ranges
- Missing values in structured columns

---

## 🛠 Cleaning Actions Performed

- Converted price column to numeric format
- Extracted numeric rating values
- Extracted publishing year using regex
- Converted reading age ranges to minimum numeric value
- Preserved informative missing values
- Validated data distribution after cleaning

---

## 📊 Final Result

✔ Delivered an analysis-ready dataset  
✔ Structured for Exploratory Data Analysis (EDA)  
✔ Suitable for machine learning modeling
